Welcome to the Docx to PDF Converter

This simple web application allows you to easily convert DOCX files into PDF format.

How to Convert a DOCX File to PDF

  1. POST Request: Make a POST request to the following endpoint:
      POST /convert

Request Body: Attach the selected DOCX file as the upfile key in the request body. This file will be converted to PDF.

  1. Conversion Process: After making the POST request, our converter will automatically process the DOCX file and convert it into a PDF file.
  2. Download the PDF: Once the conversion is complete, you will receive the converted PDF file as a response from the server.

Happy converting!
